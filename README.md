# ERA_web
A minimalist calendar web app focused on clarity, time awareness, and intentional navigation.

ERA is a single-page calendar web application designed around simplicity, flexibility, and visual calm.  
It removes unnecessary complexity and focuses on what matters most — understanding and navigating time without friction.

---

## Philosophy

ERA follows a simple principle: **time should be effortless to navigate**.

- Clean, distraction-free interface
- Clear visual hierarchy for dates and events
- Fast, predictable interactions
- Zero learning curve

The interface stays out of the way so your focus stays on time and planning.

---

## Design System

- **Theme:** Minimal Utility / Modern UI
- **Modes:** Light mode + Dark mode
- **Backgrounds:** Soft neutrals (Light), deep blacks and grays (Dark)
- **Layout:** Full-screen adaptive calendar grid
- **Navigation:** Bottom control bar for year, month, and date jumps
- **Components:** Rounded cards with subtle separation
- **Typography:** Clean sans-serif with high readability
- **Motion:** Minimal, purpose-driven transitions

---

## Features

- Navigate to any year, month, or date
- One-tap return to today
- Built-in global events:
  - Indian (Hindu) festivals
  - International / American holidays
- Visual event indicators on calendar dates
- Click a date to view its events
- Add personal notes and custom events
- Live display of today’s date, day, and time
- 12-hour / 24-hour time format toggle
- Manual light / dark mode switch
- Persistent data using `localStorage`
- Fully responsive single-page layout

---

## Privacy by Design

All data is stored locally in your browser.

- No accounts
- No tracking
- No backend
- No external APIs required

Your data never leaves your device.

---

## Tech Stack

- **HTML5** — semantic single-page structure
- **CSS3** — responsive layout and theming
- **Vanilla JavaScript** — calendar logic and state handling
- **localStorage** — client-side persistence

No frameworks. No dependencies.


---

## Use Cases

- Personal planning and time tracking
- Academic or portfolio project
- Learning JavaScript date handling
- Minimal productivity tooling
